# 对照镜 (Comparison Mirror) — 修改质量管控与逐项裁决工具

> 态控工具分层体系中的**必用工具**。将比对从"事后手动查看"变为"嵌入工作流的自动质量闸门"。

## 定位

对照镜拦截任何对**已确认内容**的修改尝试，强制执行原文与修改版的逐字差分比对，生成可视化裁决清单，逐项由用户确认/拒绝/编辑后，自动合并生成最终版本。

四工具闭环中的位置：

```
proofreader → whiteboard → 对照镜 → xiaoshan
（入口拦截）   （中游隔离）   （质量闸门）   （出口编辑）
```

## 核心创新

| | Word 追踪修订 | 传统 diff 工具 | 对照镜 |
|---|---|---|---|
| 触发方式 | 用户手动开 | 用户手动选文件 | **工作流关键节点自动拦截** |
| 责任归属 | 中立，不标来源 | 中立 | **AI 修改默认标为 AI 责任** |
| 输出形式 | 流式操作日志 | 差异报告（只读） | **逐项裁决清单（可操作）** |
| 闭环能力 | 手动接受/拒绝 | ❌ 需离开界面手动改 | **裁决后自动合并替换** |
| 遗漏检测 | ❌ | ❌ | **检测标记内容被完全遗漏** |
| 深层偏离 | ❌ | 仅表层文字 | **识别相似词但意图转变** |

## 分阶段机制

### 阶段一：修改内容确定（可选，仅 AI 辅助修改场景）

```
用户："把第五段改得更有说服力"
  → 无具体替换文本 → LLM 生成修改草案 → 用户确认/编辑
用户："把第五段第三句改为XXX"
  → 有具体替换文本 → 跳过 LLM，直接精准替换
      ↑ 此步骤不调 LLM，纯确定性文本替换
```

### 阶段二：逐字比对 + 裁决清单

```
确认原文快照 vs 修改版本
  → 逐字差分（LCS/Myers 算法）
  → 遗漏检测（标记内容是否完全消失）
  → 深层偏离检测（相似词但意图转变）
  → 可视化裁决清单
      ├─ 删除差异：确认/拒绝
      ├─ 新增差异：确认/拒绝
      ├─ 修改差异：确认/拒绝/手动编辑
      └─ 遗漏项：恢复/忽略
  → 根据裁决结果自动合并 → 替换原文
```

## 静默模式

修改量低于阈值（如只改了一个标点）时，不弹 UI，但比对执行、合并执行、审计日志写入。用户可随时回溯审查。

```
修改量 < 阈值 → 静默通过（比对机器完成，呈现层降级）
  → 审计日志持久化（仅追加写入，不可篡改）
  → 用户可事后回顾/重新裁决
```

## 核心模块

| 模块 | 职责 | 执行体 |
|------|------|:--:|
| 已确认状态管理 | 自动附加 confirmed 标记（含快照哈希+时间戳） | DET |
| 修改触发检测 | 6 种触发事件监控 | DET |
| 修改内容确定 | 分支判断 + LLM 草案生成 + 用户确认 | DET + LLM |
| 精准替换执行 | 确定位置替换，不调 LLM | DET |
| 逐字差分比对 | LCS/Myers 混合算法 + 遗漏检测 + 深层偏离 | DET + 句嵌入 |
| 裁决清单生成 | 可视化差异项 + 责任标注 + 裁决控件 | UI |
| 裁决合并引擎 | 根据裁决结果自动合并原文与新版本 | DET |
| 静默模式控制器 | 增量阈值判断 + 审计日志持久化 | DET |

## 触发事件（6 种）

| # | 触发事件 | 示例场景 |
|---|---------|---------|
| a | AI 全局替换已确认内容 | "用 AI 重写这篇" |
| b | 外部文件上传覆盖 | 导入同事修改版 |
| c | 已收存条目版本迭代 | 对已存储文档发新版 |
| d | 阶段总结生成/更新 | 已标记原文片段触发总结 |
| e | 用户主动召唤 | "对照镜" |
| f | 任何来源的修改试图覆盖 confirmed=true 内容 | 系统自动拦截 |

## 责任标注

| 修改来源 | 默认责任标注 |
|---------|------------|
| AI 生成 | **AI**（红色标签） |
| 外部文件 | **外部文件**（蓝色标签） |
| 用户手动输入 | **用户**（灰色标签） |

## 与态控的对应关系

| 对照镜模块 | 态控组件 | 类型 |
|---------|---------|:--:|
| 已确认标记 | sessionCheckpoint confirmed 状态 | field_based |
| 修改事件拦截 | IN_SESSION 入口 DET 拦截 | DET |
| 逐字比对 | DET 复验（文本差分） | DET |
| 意图偏离检测 | logprobs 阈值裁决对位（语义层） | LLM + DET |
| 裁决清单 | WAITING_CONFIRM 对位 | 人 |
| 静默模式 | 降级链"软拦截→硬拦截"对位 | DET |
| 审计日志 | roomConversationLog 追加写入 | 上下文管理器 |
