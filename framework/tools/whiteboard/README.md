# 白板 (Whiteboard) — 过程隔离的对话流外协同处理工具

> 态控工具分层体系中的**必用工具**。将修改过程从对话流彻底卸载，算力消耗从 O(N) 降到 O(1)。

## 定位

白板在主对话流之外建立一个独立的**临时协同层**。用户在此层上对内容做任意次修改——不调 AI、不占 token、不污染上下文。改完后仅将最终版本送回对话流。

与小山（语义编辑）、校对哨兵（输入预校准）构成三工具闭环：
- **校对哨兵**：拦截错误于入口
- **白板**：隔离修改于中游，避免过程性消耗
- **小山**：精准编辑于出口

## 核心机制

### 过程隔离

```
对话流（主房间）                   白板（临时协同层）
─────────────────                  ────────────────
"帮我写一篇公告"                     
  ↓ AI 生成初稿                    
  ↓ 用户："白板"                     → 激活白板，初稿上板
                                     → 用户改 10 次字词（非 AI，0 token）
                                     → AI 整体替换 2 次（调模型）
                                     → 冲突裁决 2 次（手动改动被标记）
                                     → 最终确认
  ← 回传最终版                       
追加 1 条消息                       
                              
Token 消耗：3 次 AI 调用           Token 消耗：0（中间修改不回流）
上下文占用：最终版 + 1 条记录       上下文占用：0
```

### 双操作协议

| 操作类型 | 驱动方式 | 范围 | AI 调用 | Token | 上下文 |
|---------|:--:|------|:--:|:--:|:--:|
| 用户局部修改 | 非 AI | 仅局部 | 0 | 0 | 0 |
| AI 整体替换 | AI | 全文覆盖 | 1 | 输出 token | 0 |

### 冲突裁决

过程隔离导致 AI 对用户的手动修改"不知情"——当用户先手动改了几个词、再让 AI "重写全文"，AI 生成的新版本基于的是修改前的旧语义。冲突裁决不是可选偏好，是过程隔离架构的必然配套：

```
用户手动改字词 → 对象标"内容篡改标记"
用户："重写这篇" → AI 整体替换引擎检测到标记
  → 弹出裁决提示："这部分你手动改过，覆盖吗？"
  → 确认 → 覆盖执行
  → 拒绝 → 保持当前版本
```

## 核心模块

| 模块 | 职责 | 执行体 |
|------|------|:--:|
| 临时界面区域 | 渲染独立二维编辑空间 | UI |
| 内容精准召唤 | "白板"唤醒 + 指定内容上板 | DET |
| 用户局部操作引擎 | 手势/指令直接修改，不调 AI | DET |
| AI 整体替换引擎 | "重写/换个风格" → 调 AI 全文替换 | LLM |
| 协同冲突裁决 | 篡改标记检测 + 用户确认弹窗 | DET |
| 画布内容回传 | 最终版序列化为用户消息写入对话流 | DET |
| 临时状态管理器 | 中间操作缓存隔离，不回流传 | DET |
| 资产按需转化 | 成熟内容快照为永久产出物 | DET |
| 多格式输出与部署 | PDF/图片/在线页面 | DET |
| 多模态排版与渲染 | 字体/表格/图片排版（非 AI） | DET |
| 多模态模式切换 | 文本/表格/图片模式切换 | DET |
| 动态分屏布局 | 根据屏幕特征选最优分屏方案 | DET |

## 与态控的对应关系

| 白板模块 | 态控组件 | 类型 |
|---------|---------|:--:|
| "白板" 唤醒 | M1 口令层 EXACT_MATCH | DET |
| 临时协同层 | 独立房间（不可切换，临时生命周期） | 上下文管理器 |
| 中间操作缓存 | 独立存储，物理隔离于 roomConversationLog | contractStore 临时分区 |
| 最终版回传 | EXECUTING 写入 + outputRegistry 注册 | DET |
| 冲突裁决 | DET 复验（篡改标记检测） | DET |
| 资产转化 | outputRegistry 注册为永久产出物 | DET |

## 算力收益

| 场景 | 无白板 | 有白板 |
|------|--------|--------|
| 改 10 次字词 + AI 重写 2 次 | 12 次 AI 调用，12 轮上下文 | **2 次 AI 调用，1 轮上下文** |
| 用户反复微调一段文案 | 每次"再改一下"调一次 AI | **0 次 AI 调用，纯本地编辑** |
