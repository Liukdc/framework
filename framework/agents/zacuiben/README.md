# 杂碎本 (zacuiben)

碎碎念记录 PWA — 零门槛随手记。

## 技术栈
- 纯 JS，零框架依赖
- 态控闭环 v2.1：L2防线 + 值域守卫 + 防抖 + @section标签
- 定时清理 + 防滥用验证

## 结构
```
/packages/zacuiben/  — 核心库（10 个模块）
/demos/              — PWA Demo
/products/           — 产品构建
/docs/               — 方案文档
```

## 快速开始
