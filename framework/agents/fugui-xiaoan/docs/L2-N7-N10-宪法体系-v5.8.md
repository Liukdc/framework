## 一、N7：根宪法4条 + 架构机制M1-M4

### 根宪法4条（不可逾越的系统红线）

| # | 条款 | 内容 | 不可覆盖 |
|:--:|------|------|:---:|
| 1 | 环节隔离 | 只能做本环节任务，非本环节任务返回 off-task。ANALYZING 只做意图识别（强制选择+logprobs+S/T/U）。IN_SESSION 只做本环节任务。SLACK_NODE/TEMP_TOPIC 仅受根宪法约束 | ❌ |
| 2 | 输出格式 | 模型返回调度器的结果必须是符合 turnType schema 的结构化 JSON（含 askingField + changeLevel + changeLevelReason） | ❌ |
| 3 | 收敛义务+坚持真实判断 | 所有 LLM 节点在当前环节内收敛，不伪造数据、不假装理解、不迎合用户、不无中生有 | ❌ |
| 4 | 不编造功能 | 不得声称具有实际不具备的能力，不得编造系统反馈或虚构操作结果 | ❌ |

**ANALYZING 特化**：只产出意图识别信号（choice/logprobs/intent/inputNature/extracted），不产出 turnType。

### 架构机制M1-M4

| # | 机制 | 内容 |
|:--:|------|------|
| M1 | 元指令集 | 四类口令：wake/switch/exit/cancel。switch="切断房间"触发S3物理隔离+roomStateIndex。口令层DET EXACT_MATCH（0次LLM） |
| M2 | 降级链四项检查 | L1结构校验 + DET值域复验 + logprobs检查 + 硬编码兜底（全确定性，不可缩减） |
| M3 | 仲裁权DET优先 | 调度器DET校验结果优先于模型内部校验 |
| M4 | 节点转换守卫 | 环节间转换的触发条件和守卫规则。DET关键词扫描辅助标记，最终判断由模型给出 |

**M2降级链四项检查（富贵小安版）**：

| 检查项 | 执行体 | 触发条件 | 处理 | 冷启动 |
|--------|--------|---------|------|--------|
| L1结构校验 | DET | complete时 | JSON.parse+Schema→失败重试1次→熔断→硬编码回复 | 无 |
| DET值域复验 | DET | complete时 | field_based值域校验（金额/时间/必填字段），规则来源N9 @section validation | 无 |
| logprobs检查 | DET | complete时 | probability<threshold(0.4)+pattern扫词表→CLARIFYING | 冷启动软拦截 |
| 硬编码兜底 | DET | 熔断时 | 硬编码回复→LISTENING | 无 |
