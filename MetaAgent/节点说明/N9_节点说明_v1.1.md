# N9 节点说明（元智能体 v1.1）
**版本**：v1.1 | **基于母本**：N9 节点说明 v1.3 / MetaAgent L2 v2.0

## 一、定位
N9 为每个 IN_SESSION 子类型编写一份环节宪法——16个@section标签按taskType选择正确的必填/省略规则。元智能体有16个子类型（P0/N1-N15），全部topic_based。

## 二、topic_based宪法结构（13个必填标签）
@constitution / @section taskType / @section convergence / @section role（嵌入规则5） / @section topic-boundary（嵌入规则7） / @section validation（结构化JSON，无格式约束时rules为空数组但标签不可省略+声明needSemanticExtensionCheck） / @section ask-rules（含场景特定追问示例） / @section completion / @section modification / @section off-task-detection（嵌入规则7） / @section giveup / @section output-schema（topic_based含changeLevel+changeLevelReason+topicId） / @section domain-rule-priority（默认medium）

topic_based省略标签：@section field-rules / @section parsing

## 三、产出物
16份环节宪法 + 宪法间一致性检查报告
