## 一、状态转移图

```mermaid
stateDiagram-v2
    [*] --> IDLE
    IDLE --> LISTENING : wake口令
    LISTENING --> ANALYZING : 用户输入

    ANALYZING --> IN_SESSION : intent∈{P0,N1-N15}(topic_based)
    ANALYZING --> IN_SESSION_N11 : intent=N11(field_based)
    ANALYZING --> IN_SESSION_N12 : intent=N12(field_based)
    ANALYZING --> IN_SESSION_other : intent=other
    ANALYZING --> WAITING_CONFIRM : topic_based+topic匹配需确认
    ANALYZING --> CLOSING : intent=exit口令

    IN_SESSION --> EXECUTING : complete(经DET复验,N11/N12)
    IN_SESSION --> LISTENING : complete(topic_based,event追加,经EXECUTING追加event)
    IN_SESSION --> CLARIFYING : validation_failed
    IN_SESSION --> ANALYZING : off-task
    IN_SESSION --> LISTENING : giveup

    CLARIFYING --> IN_SESSION : 用户修正后
    WAITING_CONFIRM --> IN_SESSION : confirm(topic)
    WAITING_CONFIRM --> ANALYZING : reject(topic→重新匹配)

    EXECUTING --> LISTENING : success

    CLOSING --> IDLE : 正常退出

    note right of ANALYZING
        14个topic_based intent
        + 2个field_based(N11/N12)
    end note
```
